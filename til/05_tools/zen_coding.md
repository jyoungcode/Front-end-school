# Zen-coding

- div#name
`<div id="name"></div>`

- div.name
`<div class="name"></div>`

- table>tr>td

```html
<table>
	<tr>
		<td></td>
	</tr>
</table>
```

- p.one+p.two

```html
<p class="one"></p>
<p class="two"></p>
```

- p[title]
`<p title=""></p>`

- a[href="#"]
`<a href="#"></a>`

- p.title|e  
`&lt;p class="title"&gt;&lt;/p&gt;`

- p*3

```html
<p></p>
<p></p>
<p></p>
```

- p.name-$*3

```html
<p class="name-1"></p>
<p class="name-2"></p>
<p class="name-3"></p>
```

- ul+

```html
<ul>
	<li></li>
</ul>
```

- ul+*2

```html
	<ul>
		<li></li>
	</ul>
	<ul>
		<li></li>
	</ul>
```
