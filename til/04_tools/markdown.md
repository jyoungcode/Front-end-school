# MarkDown 작성법

- code 기입:

```css
...코드 기입..
```

- 작은코드: `<a>`
- ul: '-' 리스트 ( 계층적 들여쓰기 가능 )
- 설명문: > 옆에 짝대기
- 이미지 삽입: `![대체텍스트](../images/exam.png)`
- h1: # (자동hr생성)
- h2: ## 
- h3: ### (소제목 적합)
- h4: #### 
- h5: #####
- h6 기본회색글씨: ######

- 기울임: `*기울임*`
- 굵게: `**굵게**`
- 참조:
> [링크1][1] 과 [링크2][2]

> [1]: http://example.com/ "링크제목1"

> [2]: http://example.org/ "링크제목2"

> 한줄에 한칸씩 띄어야 한다.

- link: `[대체텍스트](http://www.example.com)`
- 수평선: '---' or '***' (3개이상 사용시 적용)
- 각주: [^1]
- 줄바꾸기: 라인끝 2개이상 스페이스 
- 문단 나누기: 사이에 한줄을 비워놓자.
- ol: 1. 목록 1. 목록 
- 설명: > > , > *

- dl,dt,dd:
> wordpress

> : 오픈소스

- 축약어:
> HTML

> *[HTML]: HyperText...
