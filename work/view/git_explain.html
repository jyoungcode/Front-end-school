소스 저장소란?

version control system.

  • (VCS) 버전관리시스템파열 변화를 시간에 따라 기록가능

  • 특정 시점을 다시 불러올 수 있다.

구체적인 이점

  • 기록을 통해 변경된 내용 공유

  • 타인이 작업한 내용을 쉽게 병합

  • 과거 상태로 복구 가능

  • 분기(브랜치)를 통해 병렬 관리

VCS는 어떤게 있는가?

  • SVN ( 아직까지 쓰지만 Git으로 넘어감 )
  • Mercurial

Git ?

  • 복잡한 Branch 관리에 적합
  • 로컬 저장소와 원격저장소의 분리
  • 빠른 속도
  • 다양한 서비스 업체
  • 다양한 보조 툴
