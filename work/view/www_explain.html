웹이 만들어진 이유

  • URL 주소로 쉽게 연결 ( 카톡, 지도 , 메일, 정보, 검색, 영상 등 )
  • Linked data 는 HTML을 통해서 각각의 데이터를 연결 시키고 또다른 데이터를 생산해낸다.
  • 사람들 머리속에는 정보의 조각이 있다. 뇌는 그 조각을 맞추려는 생각을 한다.
  • 우리는 데이터를 묶어주는 일을 한다. 그 환경이 WWW다
  • 각 컴퓨터에서 문서를 공유하고 싶으나 없었다

계속 된 진화

  • 90년대 후반 .COM 열풍 ( 웹 1.0 )
  • 구글 , 아마존, 이베이 ( 웹 2.0 ) : 상생 플랫폼, 참여유도, 공유, 개방
  • API 의 등장
  • 플리커 , 트위터, 페이스북 각각 API를 각 회사에서 제공하고 사용자는 공유하게 된다.
  • 다음(약도), 네이버(네이버검색결과), SK플래닛 11번가(쇼핑몰 개발)
  • Ajax ( 에이젝스 ) , JSON : 이제 웹기술만 가지고도 구현 가능하다.
  • 웹은 문서 공유가 아닌 애플리케이션 플랫폼 거듭, 진화
